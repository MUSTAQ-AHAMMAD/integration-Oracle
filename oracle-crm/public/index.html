How the Sync Works

Use the New Sale Push form to manually send a VendHQ sale to Oracle Fusion. The CRM applies all calculations automatically and shows you a preview before pushing.

  1. Fill in the Sale header (invoice number, date, outlet, customer type)
  2. Add line items (item, quantity, total price, tax)
  3. Add payments (type, amount)
  4. Click Preview Calculations to see all derived values
  5. Click Push to Oracle Fusion to execute the 8-step API sequence
  6. View the result steps and Oracle transaction numbers in the result panel
Oracle API sequence executed on every push
